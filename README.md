# IBM TORCS Racing Challenge

[Link to our development blog](https://ai-group-project-group-4.github.io/project-blog/)

Our project is split into two parts:
 - TORCS parallel training: Our newer approach, which is more flexible, allowing for training of RL algorithms. It was used to train our 1:26:12 SAC model.
 - TORCS single instance training: Our old approach, which we originally used to train our 1:32:30 PPO model.
