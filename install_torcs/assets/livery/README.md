# Custom `car1-ow1` livery

Place **`car1-ow1.rgb`** here (TORCS texture atlas for the OW1 car). During install, `install_torcs.sh` removes the stock `cars/car1-ow1/car1-ow1.rgb` from the TORCS data tree and copies this file in its place.

If this file is absent, the installer keeps the default TORCS skin.
